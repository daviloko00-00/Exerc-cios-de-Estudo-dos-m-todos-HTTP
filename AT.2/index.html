Menu da Calculadora

Escolha uma operação e veja como usar via query string na URL:

  • Soma (10 + 5)
  • Subtração (10 - 5)
  • Multiplicação (10 * 5)
  • Divisão (10 / 5)

Você também pode alterar os números diretamente na URL para testar:
http://localhost:8081/calculadora?operacao=soma&numUm=7&numDois=3
